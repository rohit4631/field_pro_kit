## 1.0.0

* Initial stable release of **Field Pro Kit**.
* Introduced `AppTextField` with built-in validation, 3D neumorphism, and error shaking animations.
* Added `AppDropdownField` for custom styled dropdown selection.
* Added `AppDatePickerField` for seamless date selection.
* Comprehensive global styling system via `FieldProTheme` in `ThemeData`.
* Built-in support for multiple field types (email, password, search, url, etc) with automatic validations.
